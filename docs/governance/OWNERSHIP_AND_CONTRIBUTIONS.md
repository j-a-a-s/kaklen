# Propiedad y contribuciones

Kaklen es software propietario con código fuente visible públicamente. El acceso
al repositorio no concede permiso de uso, copia, modificación, distribución ni
creación de obras derivadas.

Los issues y sugerencias pueden recibirse para evaluación. Los pull requests
externos que incluyan código requieren autorización escrita previa de `j-a-a-s`.
No se acepta código sin un acuerdo de contribución aprobado por el owner.

Quien contribuya debe poseer todos los derechos necesarios sobre el material
propuesto y poder acreditarlos. Ninguna contribución modifica la licencia
propietaria de Kaklen ni concede derechos adicionales sobre el proyecto.

Las contribuciones de código no autorizadas se cerrarán sin integrar. La
participación en una conversación o revisión no implica autorización para
contribuir código.

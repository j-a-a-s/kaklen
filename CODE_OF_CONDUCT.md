# Código de conducta

Las interacciones relacionadas con Kaklen deben mantener un trato profesional,
respetuoso y centrado en el trabajo. No se toleran acoso, discriminación,
amenazas, hostigamiento ni publicación de datos personales de terceros.

Las vulnerabilidades no deben publicarse como issues, discusiones o comentarios.
Deben reportarse mediante el canal privado indicado en [Security](SECURITY.md),
sin exponer secretos, datos personales ni detalles explotables públicamente.

`j-a-a-s`, como owner, aplica esta política y puede moderar, cerrar o restringir
participaciones que la incumplan. Las decisiones de aplicación buscan proteger
a las personas, al proyecto y a sus usuarios.
